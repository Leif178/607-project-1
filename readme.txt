Usage manual for John Batarekh 607 Project 1

This repository includes the following:
__pycache__ directory
__images__ directory
main.py
Phys_607_Project_1_Report.pdf
problems.py
readme.txt

To run the code, open the directory of this file in your console and then type main.py. In order to progress to the end of the main program you must close out of each image before getting to the next one.

If you would like to use problems.py in your own script or make adjustments to the main.py, then you can call either the Oscillator or Gamma objects which are documented in the code.
